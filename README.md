# Vanilla CV Portfolio 🎌

A minimalist, responsive, and framework-free personal portfolio web application. 

This project demonstrates a clean architectural approach by strictly separating structure, style, and logic into distinct files. It avoids bloated mainstream frameworks in favor of fast, reliable, and deeply understood vanilla technologies.

## ✨ Features
* **Custom Authentication Flow:** Client-side login system built from scratch.
* **Zero Dependencies:** Pure Vanilla HTML, CSS, and JavaScript. No React, no Tailwind, no external libraries (except FontAwesome for UI icons).
* **Responsive CSS Grid Layout:** Adapts seamlessly to mobile, tablet, and desktop screens.
* **Interactive UI/UX:** Includes password visibility toggles, loading state simulations, and error animations (shake effect).
* **Accessibility (a11y) Focus:** Semantic HTML and ARIA attributes integrated for screen readers.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and accessibility.
* **CSS3:** Custom properties (variables), Grid/Flexbox layouts, and keyframe animations.
* **Vanilla JavaScript:** DOM manipulation, event handling, and state management.

## 🚀 How to Run Locally
1. Clone the repository to your local machine.
2. Navigate to the project folder.
3. Open `index.html` directly in any modern web browser.
4. **Demo Credentials:** Use `test` for both Username and Password to bypass the login screen.
